Dyslexia Prediction Results

Our model predicts dyslexia likelihood based on survey and quiz data.

  • 0 --> Low chance of dyslexia
  • 1 -->Moderate chance of dyslexia
  • 2 --> High chance of dyslexia

Run the Prediction Model on Google Colab

Follow these steps to run the model in Google Colab:

  1. Go to Google Colab
  2. Upload the project Python notebook file (e.g., prediction.ipynb)
  3. Download and upload the dataset: download labeled_dysx.csv present in data folder
  4. Ensure the dataset is in the same folder as the notebook in Colab
  5. Run all the cells and enter your survey + quiz values when prompted
  6. Check the printed result

Note: The model is based on a Random Forest Classifier trained on combined quiz and survey results.
